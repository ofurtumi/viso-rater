# Viso Rater

Kerfi til að gefa vísindaferðum einkunnir.

Sækir upplýsingar um vísindaferðir á [nord.is](https://nord.is/bakendi/feed/)

# Uppsetning

Sett upp með `vite` og `svelte`, mælt með að nota yarn til að setja upp.

```bash
yarn
# síðan
yarn dev
```
